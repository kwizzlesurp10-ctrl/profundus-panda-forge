# ProfundusPandaForge MCP Server

**A unique, personalized Model Context Protocol (MCP) server** blending the ruthless Profundus deep-research methodology with Keith Severson's multi-agent swarms (SwarmForge, BADPANDA Orchestra, HyperAgent), creative panda forge, prompt vaults, and indie hacker tools.

Created for Local AI Integrations and personal projects (KaleidoBeats, PrideWave/Eropulse, AddictiForge, etc.).

## Quick Start

1. Clone the repo.
2. Copy `artifacts/profundus_panda_forge_mcp_server.py` from your local `/home/workdir/artifacts/` (or download the full implementation).
3. Install: `uv add "mcp[cli]"` (recommended) or `pip install "mcp[cli]"`.
4. Run: `uv run artifacts/profundus_panda_forge_mcp_server.py`
5. Configure in Claude Desktop / Cursor / VS Code Copilot (see server file header for exact JSON config).

## Key Features

- **ruthless_decompose**: Profundus-style breakdown into 15-20+ sub-questions with confidence ratings.
- **orchestrate_swarm**: Ready templates for your HyperAgent / BADPANDA / SwarmForge patterns.
- **manage_prompt_vault**: Full CRUD + search for your Seed-to-Roots and Profundus prompts.
- **generate_creative_forge**: Cyberpunk panda / Suno / visual prompt generator.
- **mrr_forecast**: Brutal realistic MRR simulator with scenarios and probabilities.
- **analyze_for_mcp**: Guidance on exposing your projects as MCP capabilities.
- Resources: projects manifest, prompt vault index, embedded Profundus methodology.

Local-first design with JSON state files for full control and privacy.
